# 💎 PROJECT AMBER: YT-DLP WEB DOWNLOADER

> [!abstract] Описание
> Веб-сервис для быстрого скачивания медиа (аналог `cobalt.tools`), использующий всю мощь `yt-dlp`. Акцент на минимализме, отсутствии рекламы и высокой скорости работы.

## 🛠 ТЕХНОЛОГИЧЕСКИЙ СТЕК

- **Backend**: Python (FastAPI) — легковесный и быстрый асинхронный сервер.
- **Core**: `yt-dlp` (библиотека) + `ffmpeg` (для сборки видео).
- **Frontend**: React (Vite) + TypeScript + Vanilla CSS (Glassmorphism UI).
- **Deployment**: Docker-контейнер для изоляции зависимостей.
